# ko-wpdm-login-override-redirect

Public release of the ko-wpdm-login-override-redirect WordPress plugin.

## Installation
1. Upload the plugin folder to /wp-content/plugins/
2. Activate in WordPress
3. Configure settings as needed

## Notes
- No secrets or API keys are stored in this repo
- Configure external services via admin or filters
